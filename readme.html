Description

Addons for Beaver Builder features professional looking, easy to use yet highly functional extensions that can be used in a Beaver Builder page builder.

Beaver Builder plugin must be activated to use this plugin. After you activate the required plugins, the elements should be available for use in Beaver Builder.

Installation

  1. Install and activate the Beaver Builder page builder.
  2. Make sure you deactivate the free plugin Addons for Beaver Builder prior to installing the premium version.
  3. Unzip the downloaded addons-for-beaver-builder-pro.zip file and upload to the /wp-content/plugins/ directory or upload the plugin zip with the help of Plugins→Installed Plugins→Add New button. Activate the plugin through the 'Plugins' menu in WordPress.
  4. For Portfolio element, install and activate the optional plugin Portfolio Post Type plugin. The Portfolio addon can be built using custom post type registered by this plugin.
  5. For further help, visit the plugin documentation located at Beaver Builder Addons→Documentation in the WordPress admin.

Optionally, if you have premium version installed, you can import the sample data that replicates the demo site for you by importing the file sample-data.xml file located in the plugin directory. The import option is available under Tools → Import in WordPress admin.

Frequently Asked Questions

Does it work with the theme that I am using?

Our tests indicate that the elements work well with most themes that are well coded. You may need some minor custom CSS with themes that hijack the styling for heading tags by using !important keyword.

The demo site is best recreated with a theme that supports a full width page template without sidebars. The elements can still be used in the pages of default template.

How to enable the dark version for any element?

In Beaver Builder page builder, add a row wrapper for the element, edit row and navigate to the 'Advanced' tab of the 'Row Settings' popup window. Scroll down to the bottom of this tab to the 'CSS Selectors' section and input class ‘labb-dark-bg’ in the 'Class' field to activate the dark version of an element.

My portfolio does not show any items.

Pls install and activate the Portfolio Post Type plugin to enable custom post type Portfolio.
